### 安装 `$ npm i yrui`.

## 运行demo

1. 创建项目:

- 
`
$ git clone https://github.com/ahyiru/yrui-demo.git
`

或者

- 
`
$ npm i -g yrui-init; yrui-init appName;
`

2. 安装依赖:
`
$ yarn install 或 cnpm i
`

3. 运行项目:
`
$ npm(yarn) start 		//默认端口8080
`

4. 打开浏览器，访问: [http://localhost:8080/](http://localhost:8080/)
